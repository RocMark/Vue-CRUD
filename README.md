## 使用
```
  // 建立專案環境
  npm i 

  // 啟動 Vue 專案
  npm run serve
  
  // 開啟 json-server
  npm run json-server
```

## 功能
- 全域 SCSS 檔案
- 全域 axios
- JSON Server
- Git Commit 模板
- BluePrint 藍圖建立檔案
